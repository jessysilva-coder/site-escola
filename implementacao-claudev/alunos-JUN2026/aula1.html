Visão geral

1 Ferramenta — 3 Poderes Diferentes

Claude Chat

Pensar. Ideal para conversar, organizar ideias, escrever, analisar informações e estruturar raciocínios.

Claude Cowork

Fazer. Ajuda a trabalhar com arquivos, projetos, conectores, rotinas e tarefas mais operacionais.

Claude Code

Construir. Voltado para código, desenvolvimento, automações técnicas, sites, apps e integrações.

Aplicação

Quando usar cada ferramenta

Claude Chat
  • Você precisa organizar ideias, escrever textos, criar roteiros ou revisar uma comunicação.
  • O objetivo é pensar melhor antes de executar.
  • A tarefa não depende de muitos arquivos, conectores ou execução técnica.
  • Você precisa executar ações em arquivos, pastas, e-mails ou sistemas.
  • Você precisa criar um sistema, site, automação ou trabalhar com código.
Claude Cowork
  • Você quer trabalhar com arquivos, pastas, documentos, notas fiscais, e-mails ou rotinas.
  • Você precisa que o Claude execute tarefas com mais autonomia e contexto.
  • Você quer reaproveitar instruções, projetos, conectores e configurações.
  • A tarefa é apenas uma pergunta rápida.
  • Você ainda não revisou permissões, arquivos e conectores envolvidos.
Claude Code
  • Você precisa criar sites, apps, páginas, automações, scripts ou integrações técnicas.
  • O projeto exige estrutura de arquivos, testes e manutenção.
  • Você quer construir uma solução completa, não apenas receber uma resposta.
  • Você precisa apenas escrever, resumir ou analisar algo simples.
  • Você não quer lidar com arquivos, terminal, permissões ou estrutura técnica.

Modelos de pensamento

Qual modelo usar?

💡

Pode parecer que um modelo mais poderoso "se esforça menos" — mas é o contrário. Opus processa mais, raciocina mais fundo e custa mais tokens por tarefa, por isso tem maior impacto nos seus limites. Haiku é mais leve e econômico justamente por ser mais direto e rápido.

🔥

O esforço controla o quanto o Claude “se dedica” para formular a resposta, independente do modelo. É como regular a potência do fogão: você pode usar chama baixa, média ou alta conforme o prato.

Configuração potente

A combinação mais potente é esforço alto + modelo com pensamento ativado. Ela é ótima para problemas complexos, mas consome mais do seu limite de uso.

Uso recomendado no dia a dia

Para o cotidiano da escola — criar páginas, montar textos e responder alunos — o esforço médio sem pensamento já entrega muito bem. Guarde a configuração máxima para debugging difícil ou análise estratégica mais densa.

Principais elementos do Claude Cowork

Lógica dos elementos do Claude Cowork

O Cowork organiza o trabalho em camadas: você conversa, cria contexto, visualiza entregas, delega tarefas e automatiza rotinas.

💬

Chat

Entrada rápida para conversar, pedir ideias e iniciar tarefas.

📁

Projetos

Contexto persistente com arquivos, instruções e padrões.

🧩

Artefatos

Entregas visuais renderizadas para revisar e iterar.

📨

Despacho

Tarefas enviadas para execução em segundo plano.

🔁

Rotina

Execução automática por agendamento ou gatilho.

Memória e contexto

Por que contexto é tão importante?

🧠

Contexto é a memória de trabalho da IA durante uma tarefa. É como a mesa onde o Claude coloca todos os materiais necessários: instruções, arquivos, mensagens anteriores, regras, exemplos e objetivos. O Claude suporta janelas de contexto grandes — útil para documentos longos, comparar arquivos e projetos com várias etapas.

Boas práticas

📁 Criar um Project por tema

Um projeto para cada aula, cliente ou campanha

📝 Escrever instruções claras

Explicar objetivo, público, estilo e limites

🔀 Separar conversas por tarefa

Roteiro em uma conversa, revisão em outra

🏷️ Nomear arquivos corretamente

Facilita a leitura e organização pelo modelo

📋 Fazer resumos de decisão

Ajuda a manter continuidade entre sessões

⏸️ Pedir confirmação antes de executar

Principalmente ao usar o Cowork

🔄 Atualizar o contexto

Avisar quando algo mudou no projeto

✅ Encerrar assuntos antigos

Deixar claro quando uma tarefa terminou
