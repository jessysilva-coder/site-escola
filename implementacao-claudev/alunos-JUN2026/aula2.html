Antes de começar

Segurança dos dados

🔐

Antes de usar qualquer recurso de I.A dentro do Excel, pense nos dados que serão compartilhados. O ideal é trabalhar com bases de estudo, dados fictícios ou informações sem dados sensíveis quando o objetivo for testar prompts e aprender o fluxo.

Uso do suplemento oficial dentro do Excel

Quando usamos o suplemento oficial do Claude dentro do Excel, ele funciona dentro do ambiente do Office instalado na máquina e fica vinculado à conta utilizada. Ainda assim, a regra principal continua sendo: use apenas arquivos que você tem autorização para processar e revise políticas internas antes de usar bases reais.

Evite dados sensíveis

Não use bases reais com CPF, dados financeiros, informações de clientes ou dados internos sem autorização.

Use cópias de teste

Antes de aplicar qualquer prompt na base original, teste em uma cópia do arquivo.

Revise o resultado

O Claude acelera o trabalho, mas a validação final continua sendo responsabilidade de quem executa a tarefa.

Parte 1

Modelagem de dados

🧹

Na modelagem, o primeiro passo é analisar a estrutura da base e identificar quais colunas precisam de padronização. Nesta aula, vamos trabalhar com nome, telefone, CEP e CPF.

Colunas usadas na base

  • Nome
  • Telefone
  • CEP
  • CPF
  • Departamento
  • Escolaridade
  • Meta
  • Últimas Férias

Nota importante

Informe ao Claude um exemplo prático do formato que a coluna inteira deve receber. Exemplo: “maria EDUARDA silva” deve se tornar “Maria Eduarda Silva”.

Parte 2

Automações

⚙️

Automação no Excel não significa apenas criar fórmulas. Também envolve reduzir erros de digitação, padronizar escolhas e facilitar a inclusão de novos registros.

Por que usar validação de dados?

Quando os dados são digitados por usuários, a validação ajuda a evitar variações como “RH”, “Recursos Humanos” e “recursos humanos” dentro da mesma coluna.

Tabelas de apoio

Listas de apoio deixam o cadastro mais simples e permitem atualizar opções sem precisar mexer diretamente na base principal.

Lição de casa

Prática para entregar autonomia ao aluno

Você deve usar o Claude para extrair informações importantes de uma base de dados com cadastro de produtos.

Da base, extraia as seguintes colunas:

  • Tipo
  • Categoria
  • Avaliação
  • Data da avaliação

Após extrair as informações, crie um breve relatório para apresentação ao gestor com 3 melhorias que podem ser realizadas no processo.

Importante: nesta atividade, o aluno deve pensar no próprio prompt. A ideia é praticar a escrita da solicitação, o contexto e a validação do resultado.
