Pré-Aula

Siga esses 3 passos antes da primeira aula para chegar pronto(a) para praticar desde o primeiro minuto.

1

Contrate o plano Claude Pro

Para acompanhar os exercícios práticos das aulas você precisa de uma conta Claude Pro ativa — o plano gratuito tem limitações que vão travar sua experiência.

De preferência, use sua conta Google para facilitar o login em todos os dispositivos.

Contratar Claude Pro
2

Acesso de administrador na máquina

Para instalar os programas do próximo passo você vai precisar de permissão de administrador no seu computador.

Se usar computador corporativo, solicite o acesso ao seu time de TI com antecedência — peça antes da aula para não perder tempo.

💡

Dica: Teste se você tem permissão tentando instalar qualquer programa. Se aparecer uma tela pedindo senha de administrador e você não tiver, entre em contato com o TI da sua empresa.

3

Baixe e instale o Claude

Você vai precisar de 3 instalações para usar o Claude em todos os ambientes que veremos nas aulas:

🖥️

Claude Desktop

Aplicativo para usar o Claude diretamente no seu computador, sem depender do navegador. Necessário para integração com Excel e Power BI.

Baixar Claude Desktop
📊

Suplemento Claude no Excel

Instala o Claude direto dentro do Excel, permitindo usar I.A. nas suas planilhas sem sair do programa.

No Excel: Inserir → Suplementos → Pesquisar "Claude"

🌐

Claude no Chrome

Extensão do navegador que permite usar o Claude em qualquer página da web, integrado ao seu fluxo de trabalho no navegador.

Ver extensão para Chrome
⚠️

Faça todas as instalações antes da primeira aula. Se tiver qualquer dificuldade, tire sua dúvida no grupo do WhatsApp com antecedência.
